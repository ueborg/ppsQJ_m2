The previous analysis used a naive \`1/L\` extrapolation of Binder crossing

The key correction: for a transition with \`y_λ = 1/2\`, the standard

Re-running with the theoretically motivated \`1/sqrt(L)\`:

| Extrapolation | φ (small-ζ fit) | χ²/dof |
|---|---|---|

| 1/L^0.7 | 0.463 ± 0.021 | 13.4 |
| **1/sqrt(L) (theory)** | **0.502 ± 0.026** | **10.7** |
| 1/L^2 | 0.385 ± 0.011 | 28.2 |

**Empirical confirmation of φ = 1/2 to 1σ.** Empirical prefactor
A = 0.96 ± 0.05 (not A ≈ 0.5 as previously assumed).

## 2. NLSM theoretical framework derived

**File**: \`theory/NLSM_FRAMEWORK.md\` (this session, full derivation).

Adapted the Le Gal & Schirò replica Keldysh framework (arXiv:2511.22506,
November 2025) to QJ-PPS. The single algebraic modification: ζ
multiplies the cross-replica vertex, leaves the non-Hermitian
replica-diagonal piece unchanged. Then applied to **two model variants**:

### Case A — two competing measurements, no Hamiltonian

- Setup: H = 0, L_j = n_j at rate γ, L̃_j = ñ_j at rate α.
- Self-dual under c↔d, γ↔α; **PPS preserves this duality**.
- Symmetry class: **D**, target SO(2N)/U(N).
- π_2 = Z allows θ-term; self-dual line at θ = π.
- Universality: D-class WZW at θ = π = **Ising CFT** (c = 1/2, ν = 1).
- **Sharpest prediction**: λ_c^A = 1/2 for all ζ.

### Case B — single measurement + free hopping (project's main model)

- Setup: H = w·hopping, L̃_j = ñ_j at rate α.
- No self-duality.
- Symmetry class: **DIII**, target SO(N).
- Weak-coupling fixed point is the **free-fermion Gaussian CFT**
  (= the project's log phase).
- Naive bare-coupling NLSM gives λ_c → 1 as ζ → 0 — wrong direction.
- **Resolution**: match NLSM UV cutoff to BdG localisation length
  ξ_ps ~ λ^{-2}. Multicritical RG running from a to ξ_ps renormalises
  the bare ζ to ζ_eff = ζ·λ^{-2}. Transition at ζ_eff ~ 1 gives
  **λ_c ~ sqrt(ζ) — empirical scaling recovered from NLSM**.

Two regimes meet at crossover around ζ ~ 0.25, consistent with the
ζ = 0.5 outlier in current data.

## 6. Methodological flag: c_eff threshold method

User mentioned small-ζ jobs that ran this week were analysed with
the c_eff = 1 threshold, giving φ = 0.201. **This is the previously-
abandoned method** (see HANDOFF.md "Methodology lesson"): c_eff = 1 is
not a universal threshold at MIPTs.

The φ = 0.201 is an artifact of the wrong analysis, NOT a
contradiction with the φ = 0.502 Binder result. If those runs stored
B_L values too, the data could be re-analyzed with Binder crossings;
otherwise it's not directly useful.

Lesson: all new runs must store B_L_mean / B_L_err per (L, λ, ζ),
not just S_n.
