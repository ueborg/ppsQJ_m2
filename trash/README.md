# Trash

This folder contains archived material that is not part of the active Doob/PPS code path anymore.

## Archived groups

- `legacy_package/pps_qj/`
  - The older simulator/model/backend/algorithm stack that predated the direct `pps_qj/exact_backend.py` and `pps_qj/gaussian_backend.py` workflow.
- `legacy_scripts/`
  - One-off comparison and proposition scripts built around the archived stack.
- `legacy_tests/`
  - Tests for the archived stack.
- `legacy_notebooks/`
  - Notebooks that imported the archived simulator/model APIs.
- `examples/`
  - Example scripts tied to the archived API.
- `generated_outputs/`
  - Older plots and validation artifacts that are no longer part of the active workflow.

The live implementation is described in the top-level `README.md` and `docs/architecture_theory_map.md`.
